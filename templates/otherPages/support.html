Introduction

Learn more about popular topics and find resources that will help you with all of your Incodaq tools.

Hot to use & Functionalities

There are two types of tools on Incodaq Tools: Free tools everyone can access by going to Home page and Locked tools where one is required to Sign in to be have access to them.


List of tools

1. Merge text lines
2. Split text lines
3. Format telephone numbers
4. Format telephone numbers from Excel - Swedish edition (Locked tool)
5. Compare two lists of data
6. Morse code tool

Tool description

1. Merge text lines tool
This tool is used for merging multiple rows into a piece of text joining rows with symbols or characters of users choice.

2. Split text lines tool
This tool is used for splitting a piece of text with symbols or characters of users choice.

3. Format telephone numbers
The idea behind this tools is to use it to fix telephone numbers to fit into Swedish or Finnish telephone number format. To use it you just need to copy your Swedish or Finnish telephone numbers - each row should hold not more than one telephone number. You also need to select between available country codes (SE or FI) - SE is pre selected.

4. Formatting telephone numbers from Excel, SE edition
Formatting telephone numbers from Excel, SE edition - does all the things just as the above tool but with the convenience of uploading Excel files from which telephone numbers will be automatically standardized to Swedish telephone standard. The fixed file will be available for download.

5. Comparing two lists tool
This tools looks at the differences two inputed lists have and shows as a result instances that are found in both lists and instances that are found only in the first list (missing in the second list).

6. Morse code tools
Morse code tool provides coding standard text to Morse code.
